# Web Contacts App

Веб-приложение с формой обратной связи, каталогом и страницей заказов.

## 📋 Описание

Проект создан в рамках домашнего задания. Реализованы 4 страницы по макетам:
- Главная
- Категории (каталог товаров)
- Заказы
- Контакты (форма обратной связи)

Все страницы используют **Bootstrap 5** для стилизации и адаптивной вёрстки.

## 🚀 Запуск проекта

1. Установи зависимости:
   ```bash
   pip install flask
Запусти сервер:

bash
python app.py
Открой в браузере:

http://127.0.0.1:5000 — главная страница

http://127.0.0.1:5000/contacts — страница с формой

🧪 Функциональность
GET /contacts — возвращает HTML-страницу с формой

POST /contacts — принимает данные из формы, выводит их в консоль

404 — кастомная страница для несуществующих маршрутов

Полноценное меню с переключением между страницами
